Elements

  • script - The script tag is the root tag of the document. Allowed child tags: name, rules
  • name - The name given to the script
  • rules - This element contains the list of rules in the script. Allowed child tags: rule
  • rule - Encloses an event and response. Allowed child tags: event, response
  • event - The event pattern of the rule. Allowed child tags: subject, op-spec, pc-spec, target
  • subject - The subject of the event pattern is either a user or user attribute. Allowed child tags: user, uattr
